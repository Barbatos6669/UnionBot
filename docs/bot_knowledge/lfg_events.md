# LFG And Event Planning

Keywords: lfg event events event board signup sign up withdraw voice channel thread timer prime general lfg shotcaller ip comp content type ping reschedule cancel

Use the event board to create organized content.

Create an LFG:

1. Go to the configured event board channel.
2. Click a colored UTC prime timer button if claiming prime time, or click
   General LFG for custom/non-prime content.
3. Prime timer buttons are restricted to Shotcaller, Senior Shotcaller, Officer,
   Captain, Commander, or Guild Leader style roles depending on server config.
4. Pick a content type if the event should ping that content role, or choose
   Skip - no ping.
5. Fill the modal:
   - Event title
   - Description
   - Comp / build requirements
   - Minimum IP, if any
   - UTC/Albion date for prime timers, or Start + duration UTC for General LFG
6. The bot posts the event in the configured LFG channel.

Join an LFG:

- Click Sign up on the event post.
- Click Withdraw if you cannot attend.
- If a comp is attached, use Pick build if you need to claim a specific build.
- Read the event thread for planning and questions.

Voice channels:

- Some events create a temporary voice channel around prep time.
- Some voice channels are restricted to signed-up members for security.
- If you cannot join the event voice, make sure you signed up first or ask the
  caller/officer.

Timer reminders:

- Prime timer days use 18/20/22 UTC, then 00/02/04 UTC on the following UTC
  date.
- Discord timestamps render locally for each viewer.
- The UTC date can differ from a player's local calendar date.

Cancellation and cleanup:

- Event creators or officers can cancel events.
- Old LFG posts and temporary voice channels should be cleaned automatically.
- If something remains stuck, ask an officer to check the event ID.
