# Albion Basics

Keywords: albion basic basics beginner new player fame ip item power spec gear tier enchantment repair death lethal nonlethal premium

Albion Online is a full-loot sandbox MMO. In lethal zones, dying usually means
dropping gear. Members should bring gear they can afford to lose unless the
event says otherwise.

Core terms:

- IP means Item Power. Higher IP usually means stronger gear, but comp, timing,
  positioning, and defensive usage matter more than raw IP alone.
- Spec means specialization. Higher spec increases the strength of gear in that
  weapon or armor line.
- Tier and enchantment affect gear strength. Example: 6.1 means tier 6 with
  enchantment 1.
- Fame is Albion experience. Fame farming builds spec and unlocks better gear.
- Lethal content includes black zone, roads, many red-zone situations, and other
  content where gear loss can happen.
- Nonlethal content is safer for learning, but it should not be treated as
  practice for every lethal habit.

General advice:

- Repair before content.
- Bring food, potions, cape, mount, and swaps if the LFG requires them.
- Read the LFG before asking repeated questions.
- If a shotcaller gives a comp or IP floor, follow it.
- If you are new, say that early so the group can place you in a useful role.

Patch-sensitive warning:

Exact item stats, weapon balance, and meta builds change. If someone asks for
current best-in-slot or current patch numbers, say the bot may not know the
latest patch and suggest checking the current comp post, in-game item text, or
an experienced caller.
