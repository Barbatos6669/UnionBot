# Registration

Keywords: register registration verify verified unverified guest screenshot character screen albion name server americas europe asia officer review not in guild home guild apply application

Registration is a security and identity step. It links a Discord user to an
Albion character so the server knows they are an Albion player.

Normal flow:

1. Go to the configured registration channel.
2. Click the button labeled "Register - Registrarse - Cadastrar-se" or similar.
3. Enter Albion character name and server: Americas, Europe, or Asia.
4. Upload a screenshot of the Albion character screen in the same channel within
   5 minutes.
5. An officer reviews and approves/denies.

Important:

- Registration is not only for HomeGuild home-guild members.
- If the Albion API cannot show a guild or alliance, that does not automatically
  mean the user is fake.
- Users can still be approved as Guest/manual review if officers are satisfied.
- Members should not upload random screenshots outside the registration flow.
  If they do, tell them to click Register again and follow the flow.

After registration:

- A home-guild member should usually receive the HomeGuild role/tag.
- Allied members may receive an alliance tag/role.
- Non-home or unknown members may be Guest depending on review.
- Alliance members do not need a HomeGuild guild application for alliance
  access. If they accidentally click the guild application button, tell them
  they are already registered as Alliance. If they are actually transferring
  into HomeGuild, they should open a help ticket or ask an officer.
- If someone wants to join the guild in-game, they should register first and
  then use the guild application flow.
