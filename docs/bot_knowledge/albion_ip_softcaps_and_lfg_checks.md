# Albion IP, Soft Caps, And LFG Gear Checks

Keywords: ip item power average ip soft cap softcap cap minimum ip min ip lfg gear check inspect build spec mastery specialization quality enchantment tier overgear undergear

Use this when members ask about IP, minimum IP, soft caps, why they cannot join,
or how to check gear before an event.

What IP means:

- Item Power is the strength rating of equipped gear. Higher IP generally
  increases base stats and ability strength on that item.
- IP comes from tier, enchantment, quality, item type, mastery, and
  specialization.
- Average IP is only a quick summary. Weapon, chest, healer weapon, and key role
  items often matter more than the average number.
- Bag, potion, food, and mount are not the core combat slots people usually mean
  when checking average combat IP.

Soft caps:

- Some content applies IP caps/soft caps. Extra IP above the cap may count less
  than normal.
- Because caps can change, UnionBot should not argue exact cap math unless the
  event post or current in-game tooltip confirms it.
- In capped content, correct build and spec efficiency can matter more than
  expensive overgear.

LFG gear-check answer:

1. Check the event post for content type, minimum IP, comp, and role.
2. Inspect weapon, chest, cape, food, potion, mount, and swaps.
3. If under the IP floor, ask the caller before signing up.
4. If the player has high IP but wrong role/build, tell them high IP does not
   fix a bad comp.
5. If the content is lethal, remind them not to bring more than they can afford
   to lose.

Common correction:

- "You do not melt people by raw IP alone. You melt people by catch, purge,
  pierce/debuff, damage buffs, burst, execute, and clean caller timing."
