# Albion New Member Learning Path

Keywords: albion new beginner learning path start starter recruit recruit advice fame spec build faction arena mists gathering economy guild discord lfg voice shotcaller

New members need a simple path, not every Albion system at once.

First steps:

- Register in Discord so officers know who they are.
- Pick content roles so they receive useful pings.
- Join beginner-friendly LFGs and say they are new.
- Learn one or two roles before buying many random sets.
- Keep lethal sets affordable.

Good early content:

- Faction warfare for group movement and caller practice.
- Crystal Arena for basic teamfight habits.
- Nonlethal or cheaper fame farming to build spec.
- Gathering/economy if they enjoy silver-making.
- Roads or black-zone content only with a group that explains expectations.

Habits to learn:

- Listen to caller instructions.
- Repair and bring consumables.
- Sign up for events instead of showing up last second.
- Ask build questions in the event thread or LFG.
- Do not bring expensive gear just to impress people.

How the bot should help:

- Give short steps.
- Point to registration, content roles, event board, and active LFGs.
- Encourage asking the caller when build details are event-specific.
