# Server Guide And Mission

Keywords: mission statement server guide start here where channel channels discord guide union guild homeguild uot alliance martlock faction lfg event board voice registration

Home Guild mission:

Home Guild is an organized Albion Online guild focused on active open-world
content, Martlock faction support, black-zone progression, and practical guild
systems. The guild values members who register properly, show up to planned
content, use Discord, learn comps, respect comms, and help the group improve.

The Discord should point members toward action:

- Register first in the registration channel.
- Apply to guild if they are not already in HomeGuild.
- Pick content ping roles so they get the right event notifications.
- Use the event board and LFG posts for planned content.
- Join the event voice channel when an LFG starts.
- Ask for help in a public help channel or mention UnionBot if officers are not
  available.

Important channel areas:

- Start Here is for rules, registration, guild applications, help tickets, and
  the server guide.
- Union Board is for official announcements, the event board, LFG posts, content
  roles, and votes.
- Union Hall is for member chat, flex posts, hall of fame, and union lore.
- Content Ops is for planning, comps, shotcalling SOP, regear policy, and regear
  requests.
- Resources is for guides, market tools, bounties, routes, and other useful
  guild information.
- Martlock Faction is for faction warfare players and faction events.
- ALLY Alliance is for alliance-level coordination and allied guild content.
- Voice and Content Voice are for normal comms and automatic event voice
  channels.

Role visibility:

Not every member sees every channel. UnionBot uses Albion API registration data
to assign identity roles:

- HomeGuild means the member is in the home guild.
- Alliance means the member is in the home alliance or an approved allied guild.
  Alliance members also usually have a role named after their guild, so staff can
  tell who they represent by checking their profile roles.
- Guest means the member is not home guild/alliance, but is allowed here for content,
  faction warfare, diplomacy, or shared events.

Most staff roles are reserved for HomeGuild members because they affect
guild operations, economy, recruiting, roster health, and policy. Shotcaller
roles are the main exception when trusted non-home guild players are allowed to run or
support content. If a member cannot see something, check registration status and
roles first.
