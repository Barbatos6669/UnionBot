# Albion Content Decision Matrix

Keywords: albion content decision matrix what should we run group size solo duo trio five ten twenty timer availability lfg faction roads gank static dungeon group dungeon ava avalonian mists hellgate corrupted arena gathering transport economy

Use this file when members ask "what can we do with this many people?" or when
availability poll results need to become content ideas.

1 player:

- Mists, corrupted dungeon, solo dungeon, gathering, transport, market, tracking,
  refining/crafting, faction scouting, fame farm prep.

2 players:

- Duo Mists, 2v2 Hellgate, duo gank/scout, Roads scouting, gathering pair,
  faction roam, small transport with scout.

3-4 players:

- Small gank, Roads green/blue objectives depending on builds, static dungeon
  entry farming, faction outposts with caution, small gathering protection,
  arena/crystal practice if enough queue roles.

5 players:

- 5v5 Hellgate, Crystal Arena, small-scale roam, static/group dungeon, faction
  outposts, Roads farming, gank group, small BZ objective pressure.

6-9 players:

- Roads, BZ roam, faction warfare, ganking, group dungeon/static, small castle
  outpost pressure, tracking/gathering protection.

10-20 players:

- Larger faction content, castle/outpost pressure, ZvZ practice, Avalonian
  dungeon prep, black-zone objectives, defensive/offensive timer content.

20+ players:

- ZvZ, hideout/territory/castle operations, faction masses, alliance objectives,
  major CTA-style content.

Decision factors:

- Is there a caller?
- Is there a healer?
- Is there catch?
- Is there scout info?
- Is the group willing to risk lethal gear?
- Is the timer already claimed?
- Is there a comp and IP floor?
- Do players have the content role/ping selected?

UnionBot answer style:

- Suggest 2-3 options, not every possible activity.
- If no caller/healer, recommend safer content or content that does not require
  heavy coordination.
- For prime timers, point to LFG/timer claim rather than ad hoc planning.
