# Albion Tools And Resources

Keywords: tools resources useful links market prices killboard builds albion online 2d albion free market murderledger albion data project killboard price check build planner

Use this when members ask what sites or tools to use for Albion research,
market prices, builds, killboards, or calculators.

Important warning:

- Third-party tools can be useful, but market data is only as fresh as the
  players uploading it.
- Prices are not a promise that an item will be available instantly at that
  value. Treat market tools as planning tools for buy orders, sell orders, and
  route decisions.
- Videos and build pages can go stale after patches. Current LFG comp notes and
  caller instructions beat old videos.

Core resources:

- Official Albion guide hub: official game guides and basic systems.
- Albion Wiki: mechanic reference, items, zones, and system pages.
- Official Albion killboard: current kill links and proof for bounties.
- Albion Online 2D: item database, calculators, map/tools, and integrated
  Murder Ledger sections.
- Murder Ledger: PvP build stats, weapon matrix, kill/death research, and meta
  trends.
- Albion Free Market: market, crafting, refining, farming, black market, build,
  map, repair, and route tools.
- Albion Data Project: market data uploader/API used by multiple community
  tools. Running a data client helps keep market prices fresher.
- Albion Online Grind: build guides, comp examples, and item price references.

How to use tools correctly:

- Use build sites for ideas, then adapt to the content and caller.
- Use killboards to review deaths: what killed you, what defensives you had,
  and whether you took the fight in a bad place.
- Use market tools to place orders and plan transports. Do not assume instant
  buy/sell prices are still true by the time you arrive.
- Use the official wiki or current patch notes for mechanics when exact values
  matter.
