# Home Guild Server Overview

Keywords: home guild unionbot server overview guild alliance example alliance uot tu help officer bot

UnionBot supports Home Guild guild operations for Albion Online.

High-level systems:

- Registration verifies Albion identity.
- LFG/event board creates organized content posts.
- Prime timer claims reserve important UTC windows.
- Event signups help track attendance and voice access.
- Bounties and rewards help direct useful guild activity.
- Market posts suggest economic opportunities.
- SSO routes keep route scouting organized.
- Surveys and analytics help officers understand member experience.
- The AI helper is a quiet fallback. It should help when officers are busy, but
  it should not overrule officers, invent rules, or interrupt active
  conversations.

Response style:

- Be concise.
- Give steps.
- Mention the likely channel if configured by the bot.
- If unsure, say so and point to an officer.
- Do not invent rules, channels, rewards, or policy.
