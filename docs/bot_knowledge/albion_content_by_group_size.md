# Albion Content By Group Size

Keywords: content group size solo duo trio five man seven man ten man zerg what can we run availability poll lfg mists corrupted hellgate roads dungeon static faction gank small scale zvz

Use this when someone asks "what can we do with this many people?" or when the
availability system needs a plain-language explanation.

Solo:

- Mists, corrupted dungeons, solo dungeons, gathering, transport, market work,
  scouting, learning arena.
- Good for learning mechanics, earning silver, and building spec.

Duo:

- Duo Mists, 2v2 Hellgates, gank/anti-gank scouting, gathering escort, small
  roads objectives if players know escape routes.

3-4 players:

- Ganking, roads roaming, faction skirmish support, small dungeon dives, scouting
  and transport protection.
- Needs discipline because there may not be enough roles to fix bad engages.

5 players:

- Standard small-scale shell: tank/catch, healer, pierce/support, 2 DPS.
- Can run 5v5 Hellgates, small faction parties, roads, static dives, gank groups,
  and black-zone roams.

6-8 players:

- Stronger roads group, ava road clap comp, static dungeon group, faction brawl,
  outpost pressure, organized ganking.
- Add support/defensive tools and avoid turning into a random pile of DPS.

9-15 players:

- Small zerg / organized brawl. Needs caller, backup engage, healers, supports,
  defensive calls, and clear regroup.
- Can contest faction objectives, castles/outposts depending on opposition, and
  heavier roads/static content.

16+ players:

- ZvZ or mass faction content. Requires stricter comp, comms, caller authority,
  and attendance tracking.

UnionBot answer rule:

- If there is an existing LFG, point members to the signup post.
- If there is no LFG and enough people are available, suggest the safest content
  that fits the number and risk level.
- Do not create a fantasy comp. Ask for caller preference if the group wants
  lethal PvP.
